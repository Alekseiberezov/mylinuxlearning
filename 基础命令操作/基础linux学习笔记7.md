# 本篇讲四个命令-grep,-wc,-echo,-tail

## grep

此命令是用来查找文件中的对应关键字的

语法： grep [-n] 关键字 路径

![](/grep-n.png)

## wc

统计文件的行数，单词数量等

语法： wc [-c -m -l -w] 文件路径

-c 统计bytes数量

-m 统计字符数量

-l 统计行数

-w 统计单词数量

![](/wc.png)

补充：管道符

|

这玩意就是将左边的命令所得的结果作为右边的输入

![](/guandao.png)

## echo

echo命令用来在命令行输出特定内容

![](/echo.png)

如果想要在后面的内容输出加`即可`（反引号）

补充两个特殊符号：

">",将左侧命令的结果**覆盖**写入右侧文件中

">>",将左侧命令的结果**追加**写入右侧文件中

## tail

tail命令用来查看文件尾部内容，跟踪文件的最新更改

tail [-f -num] linux路径

-f 表示持续跟踪（别人改了你也能看到）

-num 表示从后往前看多少行（默认十行）
