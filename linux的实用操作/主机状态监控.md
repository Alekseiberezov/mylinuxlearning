top命令

此命令用来查看CPU内存占用情况

按ctrl+c或者q键退出

![top](./top.png)

df命令

查看磁盘的使用情况

df [-h]

-h:以更人性化的方式查看

![df-h](./df-h.png)

iostat命令也可查看（yum -y install sysstat）

语法：iostat [-x] num1 num2

-x显示更多信息

num1刷新间隔

num2刷新几次

![iostat](./iostat.png)

sar命令

很复杂

sar -n DEV num1 num2

-n查看网络，DEV查看接口

![sar](./sar.png)
